# BancoBanQuito — Portal de Pagos Masivos

Portal web para empresas corporativas con acceso al servicio de pagos masivos. Permite cargar archivos CSV con nóminas o pagos en lote, programar su efectivización, consultar el estado de cada lote y descargar reportes. También integra un buzón SFTP donde las empresas pueden depositar archivos para su procesamiento automático.

Desarrollado en **TypeScript** con **Vite**.
